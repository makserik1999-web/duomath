# Metrics and Post-MVP Backlog

## Metrics to track
- Activation rate: registered users who started first lesson.
- Lesson completion rate: started vs completed lessons.
- D1 retention: users active the day after registration.
- D7 retention: users active within 7 days.
- 3-day streak rate.
- 100 XP reached rate.

## Event map
- `user_registered`
- `user_logged_in`
- `topic_opened`
- `lesson_started`
- `lesson_submitted`
- `xp_gained`
- `streak_updated`
- `achievement_unlocked`

## Post-MVP backlog
1. Adaptive difficulty by mastery.
2. Teacher and classroom roles.
3. Hints and explanation cards per question.
4. Parent progress view.
5. Mobile-first UI improvements.
